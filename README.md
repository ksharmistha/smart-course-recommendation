# PICK FOR YOU-Course Recommendation System for Students

A smart web-based platform that recommends courses and generates personalized learning paths based on a student’s skills and interests using **Graph and HashMap data structures**.

---

## Project Overview

This system helps students:
- Identify relevant courses based on their current skills
- Get a structured learning path
- Track their progress efficiently

The recommendation engine is built using **DSA concepts (Graph + HashMap)** to ensure optimized and meaningful suggestions.

---

## Core Concepts Used [DSA]

- **Graph (Adjacency List)**
  - Represents relationships between skills and learning paths
- **HashMap**
  - Maps skills to relevant courses for fast lookup

---

##  Features

### 🔹 Skill Mapping
- Users can input their existing skills and interests

### 🔹 Personalized Recommendations
- Courses are suggested based on user skills using HashMap lookup

### 🔹 Learning Path Generation
- Graph traversal (BFS/DFS) is used to generate structured learning paths

### 🔹 Progress Tracker
- Tracks completed courses and shows progress visually

##  How It Works

1. User inputs skills  
2. System checks skill-to-course mapping (HashMap)  
3. Graph traversal generates a learning path  
4. Recommendations are displayed  
5. Progress is tracked and updated
